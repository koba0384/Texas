# NEON HOLD'EM on Streamlit

Texas Hold'em inspired web card game demo for Streamlit Community Cloud.

## Local run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

- Push this folder to GitHub
- Open Streamlit Community Cloud
- Connect GitHub
- Select the repo and branch
- Set entrypoint to `app.py`
- Deploy
